# Orbital Ion Cannon — Space Exploration Edition
A Factorio 2.0 Mod

Fork of [kuxynator's Kux-OrbitalIonCannon](https://github.com/kuxynator/Kux-OrbitalIonCannon), based on [Supercheese's Orbital Ion Cannon](https://mods.factorio.com/mod/Orbital%20Ion%20Cannon).

Adds full [Space Exploration](https://mods.factorio.com/mod/space-exploration) compatibility:
- Zone-aware surface resolution (orbit → parent planet)
- SE cargo rocket launch pad support
- Per-surface cannon tracking and GUI display
- Cannon surface name migration on configuration change
